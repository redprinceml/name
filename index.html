Les Commandes De TEST TP

Accédez au routeur par la console et activez le mode d’exécution privilégié :

enable

Passez en mode de configuration :

config t

Changer le nom de routeur

hostname RouterNameExampleHere

Désactivez la recherche DNS

no ip domain-lookup

Definir un password de Mode d'execution privilige aka mode root

enable secret PasswordExampleHere

Definir un password pour acceder au console

line con 0
password PasswordExampleHere
login
exit

Definir un password pour VTY

line vty 0 4
password PasswordExampleHere
login
exit

Chiffrez tous les mot de passe en clair

service password-encryption

Configurer l'interface de routeur g0/0

interface g0/0 =>(Pour changer le mode from config to configif et configurer l interfce g0/0)
ip address 192.168.0.1 255.255.255.0 =>(pour relier l'interface g0/0 avec cette ip address)
no shutdown =>(to turn the interface ON)

Configurer une description de l'interface de routeur g0/0

interface g0/0 (Pour changer le mode from config to configif et configurer l interfce g0/0)
Description DescriptionExampleHere
end

Enregistrez la configuration en cours dans le fichier de configuration initiale

copy running-confing startup-config

Réglage de l’horloge

clock set 12:00:00 25 may 2025

Envoyez une requête ping

ping 'ip address of pc we want to send the ping'
example: ping 192.168.1.1

Afficher les informations sur le matériel et les logiciels

show version

Afficher les information d'une interfacer specifique

show interface 'InterfaceName'
